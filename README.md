## Development

This is a static site built with HTML, CSS, and vanilla JavaScript. No build process required.

To view locally:
```bash
# Using Python 3
python -m http.server 8000

# Or using Node.js (http-server)
npx http-server
```

Then visit `http://localhost:8000`
